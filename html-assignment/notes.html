Notes

5 HTML tags that I have learned:

<!DOCTYPE html>
This singular tag identifies the document type as HTML.
<p></p>
This paired tag creates a paragraph when text is placed inside.
<nav></nav>
This paired tag contains the webpage's navigation. It is placed in the site's header.
<hr>
This singular tag creates a horizontal line that separates different topics on a page.
<ul></ul>
This paired tag contains a list in which the order of the items does not matter.
